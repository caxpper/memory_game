Arcade Memory

This game is a memory matching game. You have to find two cards that match together in order to make a pair.
The cards represent old arcade games. You must pair the two cards from the same game.
There are 9 pairs total and no identical cards.

In order to be able to accomplish this game, I applied the following skills:

  • HTML5
  • CSS3
  • JavaScript
  • JQuery
  • Git
  • Animation CSS
